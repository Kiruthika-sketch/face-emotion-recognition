Face Emotion Recognition

## Description
This project detects human emotions in real-time using a webcam or IP camera.

## Features
- Real-time face detection  
- Emotion recognition (Happy, Sad, Angry, etc.)  
- Works with webcam and IP camera  

## Technologies Used
- Python  
- OpenCV  
- NumPy  

## Files
- emotion.py → Webcam emotion detection  
- ip.py → IP camera video capture  

## How to Run

1. Install required libraries:

pip install opencv-python numpy imutils facial-emotion-recognition


2. Run webcam version:

python emotion.py


3. Run IP camera version:
- Change the IP address in `ip.py`

python ip.py
